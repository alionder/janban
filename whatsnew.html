Version 2.0.16

  • Added a config option to switch debug logging on. In case of issues this log can be added to the support email.
  • Added checkboxes to select which attachments should be added to the support email.
  • Added error handling in all functions in order to have detailed information when something goes wrong.
  • Added a popup message when you drop a task in a folder when the maximum number of items is reached.

    • Version 2.0.14

  • Added a ping back to monitor if the app is actually being used. This can be switched off if you don't want this to happen.
